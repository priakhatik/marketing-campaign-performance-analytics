## Overview
This project delivers a complete **marketing performance analytics workflow** using Python, SQL, and Tableau.

## Methodology

### 1. Data Cleaning & Transformation
- Handled missing values and inconsistent formats.  
- Standardized categorical variables and normalized continuous features.  
- Derived new fields such as `TotalSpend`, `CampaignSuccessRate`, and `CustomerLifetimeValue`.  

### 2. Exploratory Data Analysis (EDA)
- Identified correlations between **age, income, and campaign response rates**.  
- Visualized engagement levels across different **marketing channels (email, social, web)**.  
- Analyzed purchase frequency, deal usage, and customer retention trends.  

### 3. Campaign Performance Metrics
- Calculated:  
  - **Click-Through Rate (CTR)**  
  - **Conversion Rate (CVR)**  
  - **Customer Acquisition Cost (CAC)**  
  - **Return on Investment (ROI)**  
- Compared performance across **channels, age groups, and spending levels**.  

### 4. Visualization & Reporting
Created a **Tableau dashboard** featuring:
- Campaign ROI by Channel  
- Top 5 Performing Segments  
- Conversion Funnel (Awareness â†’ Engagement â†’ Purchase)  
- Customer Retention and Lifetime Value
